Our Projects

We have completed 3 projects since 2x2 Pixel Team was founded:

  • How hardware works : Experimenting with Logic Gates
  • How software works : Organizing Hour of Code event
  • How websites work : Developing our team website using HTML, CSS and JavaScript

Logic Gates

Our first project was to build logic gates such as AND, OR, NOR, NOT, NAND, XOR. That project served as an introduction to learn how hardware works.

Hour of Code event

The second project was organizing Hour of Code event. Our aim was to introduce basics of coding to children. In order to achieve that goal, we chose to tutor Classic Maze puzzle. The reason behind our decision was the tasks being simple enough for children to understand, and the inclusion of characters from popular games and animation films.

We are proud to say that we have achieved our goal. Participants were excited to solve the taks and learn something new. Furthermore, some of them even asked us where they could find more puzzles to solve!
