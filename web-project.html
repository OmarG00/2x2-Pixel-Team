Purpose

This Website project is the continuation of our Hour of Code event. Hence, you can learn about what happened during the event, as well as our team through this website. Furthermore, you can also check out the Code Artist game and experiment with Snow Flake!


Creating this website helped us improve our web development skills. We hope that this website will encourage others to improve their coding and web developing skills too! As an attempt to achieve such an effect, we documented our Hour of Code event for now. The same approach will be applied to our future projects too.

Features

We have included an interactive Code Artist game generated by algorithms. That game is designed to demonstrate Snow Flake in a fun and experimental way.

How does the game work?

What you need to do is to enter a number between 1 and 10. Then, you will see a shape based on the number you chose. We encourage you to experiment and analyse the differences when you increase or decrease the number.
